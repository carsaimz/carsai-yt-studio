# Política de Segurança / Security Policy / Política de Seguridad

## 🇧🇷 Português

### Versões Suportadas

| Versão | Suportada |
|--------|-----------|
| Latest (main) | ✅ |
| < 1.0.0 | ⚠️ Best effort |

### Reportar uma Vulnerabilidade

**NÃO abra uma issue pública para vulnerabilidades de segurança.**

Reporte de forma privada via:
1. [GitHub Security Advisories](https://github.com/carsaimz/carsai-yt-studio/security/advisories/new) (preferido)
2. Email: **security@carsai.app**

**Inclua:**
- Descrição detalhada da vulnerabilidade
- Passos para reproduzir
- Impacto potencial
- Sugestão de correcção (se tiver)

**Prazo de resposta:**
- Confirmação: 48 horas
- Avaliação inicial: 7 dias
- Correcção: 30 dias (dependendo da severidade)

## 🇺🇸 English

### Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main) | ✅ |
| < 1.0.0 | ⚠️ Best effort |

### Reporting a Vulnerability

**Do NOT open a public issue for security vulnerabilities.**

Report privately via:
1. [GitHub Security Advisories](https://github.com/carsaimz/carsai-yt-studio/security/advisories/new) (preferred)
2. Email: **security@carsai.app**

**Include:**
- Detailed description
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

**Response timeline:**
- Acknowledgement: 48 hours
- Initial assessment: 7 days
- Fix: 30 days (depending on severity)
